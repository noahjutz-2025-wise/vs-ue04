# Direct Messages Server

## Protocol

- TYPE
  - 0 ReqRegister
  - 1 ReqSend
  - 2 ReqGet
  - 3 ResStatus
  - 4 ResGet
- REQID: Generated by client
- LENGTH: 0-255
- LENGTH LENGTH: 0-65535

### Request

Register request

```
+--------+--------+--------+--------+
|TYPE    |REQID   |LENGTH  |USER...
+--------+--------+--------+--------+
 ...     |
+--------+
```

Send request

```
+--------+--------+--------+--------+
|TYPE    |REQID   |LENGTH  |USER...
+--------+--------+--------+--------+
 ...     |LENGTH  |RECIP...         |
+--------+--------+--------+--------+
|LENGTH   LENGTH  |MSG...           |
+--------+--------+--------+--------+
```

Get request

```
+--------+--------+--------+--------+
|TYPE    |REQID   |LENGTH  |USER...
+--------+--------+--------+--------+
 ...     |
+--------+
```

### Response

- STATUS
  - 00 OK
  - 01 Request malformed
  - 11 Internal error

Status response

```
+--------+--------+--------+
|TYPE    |REQID   |STATUS  |
+--------+--------+--------+
```

Get response

```
+--------+--------+--------+--------+
|TYPE    |REQID   |STATUS  |AMOUNT
+--------+--------+--------+--------+
 AMOUNT  |LENGTH  |SENDER...        |
+--------+--------+--------+--------+
|LENGTH  |RECIP...         |LENGTH
+--------+--------+--------+--------+
 LENGTH  |MSG...           |
+--------+--------+--------+
```
