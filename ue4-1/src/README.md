# Direct Messages Server

## Protocol

### Request

- Request Type
  - 00 Register
  - 01 Send
  - 10 Get
  - 11 Not a type

Register Frame

- Length: 0-63

```
+--------+--------+--------+
|00LENGTH|Username|...     |
+--------+--------+--------+
```

Send Frame

```
+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+
|01000000|Username|...     |00000000|Recip.  |...     |00000000|Message |...     |00000000|
+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+
```

Get Frame

```
+--------+--------+--------+
|10LENGTH|Username|...     |
+--------+--------+--------+
```

### Response

- Status codes (ST)
  - 00 OK
  - 01 Request malformed
  - 11 Internal error

Register Frame

```
+--------+
|00ST0000|
+--------+
```

Send Frame

```
+--------+
|01ST0000|
+--------+
```

Get Frame

```
+--------+--------+--------+--------+--------+--------+--------+--------+
|10ST0000|SENDER  |00000000|RECIP.  |00000000|MESSAGE |00000000|...     |
+--------+--------+--------+--------+--------+--------+--------+--------+
```

Error Frame

```
+--------+
|11ST0000|
+--------+
```
